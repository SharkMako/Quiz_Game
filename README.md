# Quiz_Game
Проект, сделанный в качестве курсовой работы во время обучения на факультете Среднего Профессионального Образования.

Суммируя всю деятельность, связанную с данным проектом, получилось приобрести неплохой опыт базовой работы с движком Unity, а также с созданием ярких и броских "комических" интерфейсов. Однако, по итогу всей проделанной работы, были сделанны определённые (не особенно то положительные) выводы, касающиеся стабильности самого движка Unity версии 5. Периодически возникали разного рода ошибки и проблемы, которые, к моему неопытному удивлению, решались простым перезапуском, либо пересборкой проекта.
